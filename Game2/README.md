# Game2 - VPD Defender

Story-driven first-responder driving game built in plain HTML/CSS/JS.

## What changed in this rebuild
- Dispatch-style **Mission mode** with timed callouts to randomized 4-digit addresses.
- **Free Roam mode** for no-pressure practice driving.
- Expanded hazard variety: traffic, pedestrians, potholes, tree limbs, cones, trash bins, carts, oil slicks, open manholes, and bubbling soft spots.
- Improved indie/cartoon visual pass and mobile-friendly touch controls.
- Slide-out hazard tips panel plus idle hint toast to guide stuck players.

## Run
Open `index.html` in a browser.
