YouLevel Guided Instructions

Quick Start

1. Set Hero Vehicle
Open Primary Vehicle. Pick side, sprite, facing, size, and attack mode.
2. Add A Second WebP
In Asset Library, pick an asset and click the canvas to place it.
3. Convert It To Enemy
Select the layer, open Enemy, set Enemy = On, choose model/count/spawn, then enable laser or bombs.
4. Difficulty
Use per-layer controls Layer Difficulty Tier, Health Multiplier, Damage Multiplier, then set Overall Play Difficulty in Level Properties.
5. Combat + Timing
Use Combat Tuning for projectile speed, blast radius, score multiplier. Use path controls for movement timing and direction.
6. Validate + Test
Run Validate Scene, then Quick Play.
7. Import Into A Game
Use games\youlevel\tools\import-youlevel-package.cmd to choose New Game or Existing Game.

Image Workflow

Use games\youlevel\tools\youlevel-image-loader.cmd. It asks target game and creates folder structure when needed.
