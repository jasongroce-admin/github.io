YouLevel FAQ + Examples

How do I build a basic enemy wave?

Place one enemy WebP, set Enemy = On, set count to 6, spawn both sides, tier = hard, and enable raid formation = V.

How do I make a boss?

Set Layer Difficulty Tier = boss, Health Multiplier = 3.0, Damage Multiplier = 2.2, laser enabled, bomb enabled, aggressive AI on.

Can I make a non-combat scene?

Yes. Set Enemy = Off for all layers and use path/timing tools for cinematic movement only.

How does Overall Play Difficulty work?

It scales difficulty globally at runtime while preserving per-layer tuning.

How do I import into a new game folder?

Run import-youlevel-package.cmd, choose New, enter game title. The tool scaffolds files/folders and imports the level.

How do I add images to an existing game?

Run youlevel-image-loader.cmd and choose Existing Game.
