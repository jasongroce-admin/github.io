# Game 02 - VPD Defender

Arcade-style first responder chase game built with HTML5 Canvas.

## Features
- Fast-paced top-down police response gameplay
- Random traffic, pedestrians, potholes, and fallen branches
- Score bonuses (stars) and time extensions
- Increasing difficulty levels with tornado weather phase
- Keyboard and touch controls

## Run
Open `index.html` in a browser.
