## Game 01: Pipe Rescue Patrol

- Top-down cartoon game on a hidden grid.
- Randomized holes each level (real leaks, false digs, and treasure digs).
- Leak timers with rising wetness visualization.
- Patch table flow: equip a patch, then repair broken pipes.
- Brown/yellow fountain effect for exploded leaks.
- Golden Shovel treasure powerup (20 seconds of faster digging).
- 10 levels with wastewater education tips shown between levels.
- Supports desktop controls (WASD/Arrow, mouse hold, space) and touch input.
